INSTRUCCIONES
1) Reemplaza firebaseConfig en firebase.js con los datos de tu proyecto Firebase (Firestore).
2) En la consola de Firebase: Firestore Database -> Crear base de datos (modo de pruebas para empezar).
3) Abre index.html en un servidor (recomendado) o localmente. Para pruebas de múltiples dispositivos, sube a Firebase Hosting o a un servidor accesible.
4) La imagen de ejemplo está en /mnt/data/23939ea7-4c69-4d49-896c-a4797a44961e.png — se usa como logo.
5) Para producción, configura reglas de seguridad de Firestore y autenticación.
